Criei esse pequeno software para praticar e melhorar minha lógica. Sistema simples de gestão de hábitos com persistência em JSON.
___
           
           Conceitos aplicados:
   
-Tratamento de erros.
    
-Data save e Data load
    
-Funções
    
-Validações

          Funcionalidades:
   
- Criar hábitos
- Listar hábitos
- Marcar como concluído
- Ver progresso

          Tecnologias:
- Python
- JSON
  
           Objetivo:

Praticar lógica, resolução de problemas, reforçar conhecimento de funções, sintaxe básica e desenvolvimento de sistemas.
