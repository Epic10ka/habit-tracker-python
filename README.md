           Sobre:
Criei esse pequeno software para praticar e melhorar minha lógica de programação. Sistema simples de gestão de hábitos com persistência de dados em JSON.
___
           
           Conceitos aplicados:
   
-Tratamento de erros.
    
-Persistência de dados (save/load)
    
-Funções
    
-Validações

          Funcionalidades:
   
- Criar hábitos
- Listar hábitos
- Marcar como concluído
- Ver progresso

          Tecnologias:
- Python
- JSON
  
           Objetivo:

Praticar lógica, resolução de problemas, reforçar conhecimento de funções, sintaxe básica e desenvolvimento de sistemas.
